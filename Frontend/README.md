# Frontend

This folder contains the frontend application for the project.

Prerequisites
- Node 16+ and npm or Yarn

Local setup

1. Install dependencies:

```bash
npm install
# or
yarn install
```

2. Run dev server:

```bash
npm run dev
# or
yarn dev
```

Build

```bash
npm run build
# or
yarn build
```

Tests

```bash
npm test
# or
yarn test
```

Notes
- Update `package.json` scripts to match your chosen framework (Vite, Create React App, Next.js, etc.).
