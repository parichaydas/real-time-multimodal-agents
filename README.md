# Real-Time Multimodal Agents

Lightweight project scaffold for real-time multimodal agents (backend + frontend + agents).

Prerequisites
- Python 3.10+ (for Backend)
- Node 16+ / npm or Yarn (for Frontend)
- Docker (optional)

Quick start

- Backend: see `Backend/README.md` for setup and run steps.
- Frontend: see `Frontend/README.md` for setup and run steps.

Suggested workflow

1. Prepare the backend environment (virtualenv + dependencies).
2. Prepare the frontend (install npm packages).
3. Start backend and frontend locally for development.
4. Optionally containerize with Docker and deploy.

Where to look next
- `Backend/README.md` — backend developer steps
- `Frontend/README.md` — frontend developer steps
